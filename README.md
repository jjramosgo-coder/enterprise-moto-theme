# Bitácora Enterprise

Tema de WordPress a medida para **Bitácora Enterprise** / *Diario de ruta de la Enterprise*,
un blog de viajes en moto con estética editorial oscura, escrito por Juanjo y María José.

El tema (carpeta `enterprise-moto/`) es una obra **original** construida sobre WordPress,
sin partir de ningún otro tema. Incluye bloques Gutenberg propios (etapas de ruta, mapas
con OpenLayers, galerías, colecciones de viajes, etc.) y plantillas específicas para los
«cuadernos de bitácora».

## Estructura del repositorio

- `enterprise-moto/` — el tema (código GPL-3.0-or-later; ver `LICENSE`).
- `docs/` — documentación (CC BY-NC-SA 4.0).
- `bitacora-enterprise-design.md` — documento de diseño e implementación (fuente de verdad del diseño).
- `TODO.md` — backlog del proyecto.
